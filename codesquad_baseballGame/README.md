##1단계  : 간단 야구 게임 구현하기
	1. 스트라이크, 볼, 안타, 아웃 4가지를 랜덤하게 출력한다.
	2. 야구게임은 9회 말까지 총 18회 진행한다.
		- for문을 통해서 18회 게임을 진행한다.
	3. while문을 통해서 3아웃이 되지 않으면 계속해서 게임을 진행한다.
	4. if 문을 통해서 스트라이크, 3스트라이크 아웃, 볼, 4볼 출루, 안타, 아웃 6가지의 경우의 수에 대해서 처리한다.
	5. 게임의 회가 바뀌기 전까지 out, strike, ball 경우의 수를 기록한다.
	6. 총 안타 수를 계산해서 출력한다.
	
##2단계  : 팀데이터 입력 및 시합 기능 구현
	1. 데이터 입력, 데이터 출력, 시합 시작, 시합 종료 기능 구현 및 validation
	2. 데이터 입력 : LinkedHashMap을 통해서 key값은 팀이름 value값은 선수, 타율을 순서대로 입력
	3. 2번 입력한 내용 출력 2번 입력 내용이 없을 경우 내용을 입력하라는 validation 설정
	4. 시합 기존 step1의 게임에서 추가로 팀, 총 점수 구현
	5. **시간이 없어서 타율에 맞는 확률 설정 못함.
